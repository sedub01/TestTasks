Условия: задание выполняется на языке С++ с использованием библиотеки Qt
Задание:
1. Реализовать класс дерева неограниченной вложенности (каждый элемент дерева имеет как минимум поля id и name)
2. Реализовать возможность ввода новых элементов дерева, например через консоль
3. Все данные, которыми наполняется дерево, должны быть сохранены на жесткий диск
4. Произвести чтение данных и вывести информацию в консоль в таком виде, чтобы можно было понять иерархию
элементов дерева
5. Реализовать удаления, редактирование и добавление узлов в дерево

Большим плюсом будет:
1. Использование СУБД для хранения элементов дерева

Плюсом будет:
1. Использование паттерном проектирования и стандартов С++ 11,14,17
2. Соблюдение правил структурирования и масштабирования кода
3. Написание API для методов чтения, добавления, удаления, редактирования элементов дерева
